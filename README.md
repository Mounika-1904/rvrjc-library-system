## 🌍 Deployment Details

The RVR & JC College Online Library Management System is deployed using modern cloud platforms to ensure global accessibility.

### 🔹 Backend Deployment
- Hosted on **Railway**
- Built using **Node.js + Express.js**
- Connected to **PostgreSQL Cloud Database**
- Environment variables configured for secure database access
- CORS enabled for cross-origin communication
- Production URL:
  https://rvrjc-library-system-production.up.railway.app

### 🔹 Frontend Deployment
- Hosted on **Vercel**
- Built using **HTML, CSS, Bootstrap, and JavaScript**
- Connected to Railway backend using production API URL
- Fully responsive and globally accessible
- Production URL:
  https://rvrjc-library-system.vercel.app

### 🔹 Deployment Workflow

1. Source code pushed to GitHub repository.
2. Railway connected to GitHub for backend deployment.
3. Environment variables (DATABASE_URL, JWT_SECRET) configured.
4. Railway auto-deployed backend service.
5. Frontend API base URL updated to production backend URL.
6. Vercel connected to GitHub repository.
7. Frontend deployed as a production-ready static site.
8. Final system tested in production environment.
